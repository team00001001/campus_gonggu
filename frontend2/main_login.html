How it works

어떻게 사용하나요?

01
✏️

공구 개설 또는 참여

원하는 물건의 공구방을 직접 열거나, 이미 열린 공구에 참여하세요.

02
💬

채팅으로 소통

인원이 모이면 채팅방에서 수령 장소와 시간을 조율하세요.

03
📦

물건 수령 확인

물건을 받으면 수령 확인 버튼을 눌러 공구를 성공적으로 마무리하세요.

📖 단계별 이용 가이드

1
🏠 메인 페이지
검색창에 원하는 물건을 입력하거나, 카테고리를 클릭해 공구를 찾아보세요.
🔍 검색창 입력 후 Enter
2
📋 공구 목록 페이지
원하는 공구 카드를 클릭해 상세 페이지로 이동하세요.
공구 카드 클릭 👆
3
🛍️ 공구 상세 페이지
공구 정보를 확인한 후 하단 버튼을 눌러 참여하세요.
이 공구에 참여하기
4
💬 채팅방
채팅방에서 방장과 수령 장소·시간을 조율하세요.
채팅방 가기 (참여중)
5
🛍️ 공구 상세 페이지
물건을 받은 후, 상세 페이지로 돌아와 수령 확인 버튼을 눌러 공구를 완료하세요!
📦 물건 수령 확인
1
🏠 메인 페이지
로그인 후 화면 오른쪽 하단의 ✏️ 버튼을 클릭하세요.
✏️ (오른쪽 하단 버튼)
2
📝 공구 작성 페이지
상품명, 가격, 모집 인원, 수령 장소 등을 입력하고 공구를 올리세요.
공구 올리기 🚀
3
🛍️ 공구 상세 페이지
인원이 모이면 채팅방에서 참여자들과 소통하며 공구를 진행하세요.
💬 채팅방 들어가기 (방장)
4
🛍️ 공구 상세 페이지 (방장)
물건이 도착하면 참여자 목록에서 각 참여자의 수령을 확인해주세요.
참여 확인 ✓
5
🏆 완료
모든 참여자가 수령 확인을 하면 공구가 최종 성공 처리되고 신뢰도가 올라갑니다!
🎉 공구 성공!
